# Audio Recorder

A simple, lightweight web-based audio recorder.

## Features

- **Record Audio**: Capture audio directly from your microphone.
- **Playback**: Listen to your recordings with a visual waveform representation.
- **Visual Waveform**: See real-time audio visualization during playback and recording.
- **Save/Load**: Export your recordings as `.wav` files and load existing ones.
- **Transport Controls**: Start, stop, play, erase, and skip to the beginning/end of the buffer.

## Getting Started

Simply open `recorder.html` in any modern web browser that supports the Web Audio API.

### Usage

1. Click **OFF** (which changes to **ON**) to enable microphone access.
2. Press the **Record (●)** button to start recording.
3. Press **Stop (■)** to stop recording or playback.
4. Press **Play (▶)** to listen to your recorded audio.
5. Use the **Save** button to download your recording as a `.wav` file.
6. Use the **Load** input to import a `.wav` file for playback.
